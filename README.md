# Linear Regression Projects

This repository contains machine learning projects built using Linear Regression.

## Projects

### 1. Car Risk Prediction
Predicts car risk level based on different vehicle features.

#### Features
- Data preprocessing
- Linear Regression model
- Model evaluation
- Prediction system

#### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

### 2. Salary Prediction
Predicts salary based on experience and other factors.

#### Features
- Data cleaning
- Training Linear Regression model
- Data visualization
- Salary prediction

#### Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```
